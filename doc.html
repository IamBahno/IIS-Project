Instalace

Stručně popište: Instalace na server:
  • ./setup.sh - nainstaluje všechny potřebné balíčky a závislosti, očekává se verze pythonu 3.7, výšší verze budou taky fungovat
  • source venv/bin/activate - aktivuje python vývojové prstředí
  • python init_db.py - inicializuje databazi na linku definovanem v "app/config.py"
  • gunicorn run:app - spustí aplikaci
Instalace na lokálně:
  • ./setup.sh - nainstaluje všechny potřebné balíčky a závislosti, očekává se verze pythonu 3.7, výšší verze budou taky fungovat
  • source venv/bin/activate - aktivuje python vývojové prstředí
  • v "app/config.py" zakomentujte řádek s connection stringem a odkomentujte řádek podtím (dle komentářu vně) abyste místo vytváření PostgreSQL databáze na serveru vytovřilia připojovali se k lokální SQLLite databízi
  • "python init_db.py"
  • ./run.sh - spustí aplikaci
